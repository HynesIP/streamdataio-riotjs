#Introduction to Riot.js

This branch contains the code of the [first post](http://streamdata.io/blog/exploring-riot-js-part1/) of our series around [Riot.js](http://riotjs.com/).

#Starting the sample

In a terminal, run those two commands:
- `npm run install`
- `npm run serve` (live-edit)

A browser should be opened at: `http://localhost:3000/app/`
